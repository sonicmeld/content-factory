# Project Identity

Content Factory is NOT:

- AI Chat Platform
- AI Assistant
- Agent Framework
- Coding Assistant
- Workflow Builder
- Video Renderer

Content Factory IS:

- Multi Channel YouTube Manager
- Asset Factory
- Prompt Factory
- Metadata Factory
- Upload Queue Manager
- YouTube Publisher

Video creation happens outside the system.

The system never generates final videos.

The system manages:

- Assets
- Metadata
- Uploads
- Scheduling
- Publishing
